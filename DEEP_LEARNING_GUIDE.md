# 🧠 深度学习教程使用指南

## 📖 简介

本项目提供纯 Python 实现的深度学习教程，不依赖 TensorFlow/PyTorch 等框架，帮助您深入理解深度学习的核心原理和底层实现。

## 🚀 快速开始

### 方法1: 交互式菜单（推荐）

```bash
python main.py
```

进入交互式菜单，可以浏览和运行所有教学模块。菜单提供：
- 动态发现所有可用的教学模块
- 数字选择快速运行
- 模块详细信息查看
- 友好的用户界面

### 方法2: 命令行模式

```bash
# 查看帮助
python main.py --help

# 列出所有可用模块
python main.py --list

# 直接运行指定模块
python main.py fundamentals
python main.py cnn
python main.py rnn
```

### 方法3: 直接运行模块文件

```bash
# 深度学习基础
python deep_learning_fundamentals.py

# 卷积神经网络
python deep_learning_cnn.py

# 循环神经网络
python deep_learning_rnn.py

# 实践练习
python deep_learning_exercises.py

# 高级主题
python deep_learning_advanced.py
```

## 📚 学习内容

### 1. 深度学习基础 (`deep_learning_fundamentals.py`)
- **深度学习概念**: 什么是深度学习，与传统ML的区别
- **主要架构**: MLP、CNN、RNN、Transformer等
- **训练技巧**: 批归一化、Dropout、优化算法
- **迁移学习**: 预训练模型的使用
- **深度学习框架**: TensorFlow、PyTorch、JAX介绍
- **学习路线图**: 系统性的学习规划

### 2. 卷积神经网络 (`deep_learning_cnn.py`)
- **CNN原理**: 卷积、池化、特征提取
- **操作演示**: 卷积计算、边缘检测
- **特征层次**: 从边缘到复杂对象的学习过程
- **经典架构**: LeNet、AlexNet、VGG、ResNet等
- **应用领域**: 图像分类、目标检测、分割
- **训练技巧**: 数据增强、模型设计原则

### 3. 循环神经网络 (`deep_learning_rnn.py`)
- **RNN原理**: 序列建模、记忆机制
- **LSTM/GRU**: 解决梯度消失问题
- **应用演示**: 序列预测、文本分类、时间序列
- **注意力机制**: Attention的基本概念
- **实际应用**: NLP、语音识别、时间序列分析
- **训练技巧**: 梯度裁剪、序列处理

### 4. 深度学习练习 (`deep_learning_exercises.py`)
- **练习1**: 感知机实现（理解线性分类器）
- **练习2**: 多层感知机解决XOR问题
- **练习3**: 迷你CNN实现（卷积操作）
- **练习4**: 简单RNN实现（序列处理）
- **练习5**: 梯度下降可视化
- **练习6**: 反向传播手工计算
- **学习路线图**: 分阶段的实践计划

### 5. 高级主题 (`deep_learning_advanced.py`)
- **生成对抗网络(GAN)**: 对抗训练机制
- **Transformer架构**: 注意力机制详解
- **自监督学习**: 无标签数据学习
- **图神经网络**: 图结构数据处理
- **强化学习**: 智能体学习基础
- **联邦学习**: 分布式隐私保护学习
- **模型压缩**: 高效部署技术
- **可解释AI**: 理解模型决策过程
- **未来趋势**: 最新发展方向

## 💡 学习建议

1. **循序渐进**: 不要跳跃学习，按照顺序逐步掌握
2. **动手实践**: 每个概念都要亲手实现一遍
3. **理解原理**: 重点理解算法的工作原理，而不是记住代码
4. **多做实验**: 修改参数，观察结果变化
5. **总结反思**: 学完每个主题后进行总结

## 🔧 故障排除

- 终端支持UTF-8编码

### 模块导入问题
如果提示"深度学习模块未找到"：
- 确保所有深度学习文件都在同一目录
- 检查文件名是否正确
- 运行 `python test_deep_learning.py` 进行诊断

如果程序运行异常：
- 确保使用Python 3.6+
- 检查文件完整性
- 尝试单独运行各个模块
